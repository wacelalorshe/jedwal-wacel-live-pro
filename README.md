# مشروع وسيل لايف برو - بث المباريات

مشروع ويب متكامل لعرض وإدارة جدول المباريات مع روابط البث المباشر.

## المميزات

- ✅ عرض المباريات بشكل منظم وجذاب
- ✅ لوحة تحكم آمنة بإدارة المصادقة
- ✅ إضافة وتعديل وحذف المباريات
- ✅ ربط مباشر مع Firebase للتحديث الفوري
- ✅ تصميم متجاوب يعمل على جميع الأجهزة
- ✅ دعم اللغة العربية والاتجاه من اليمين لليسار

## الملفات المطلوبة

- `index.html` - الصفحة الرئيسية لعرض المباريات
- `admin.html` - لوحة التحكم لإدارة المباريات
- `firebase-config.js` - إعدادات وتكوين Firebase
- `styles.css` - ملف التنسيقات المشتركة

## متطلبات التشغيل

1. استضافة ويب تدعم JavaScript
2. اتصال بالإنترنت لتحميل مكتبات Firebase

## إعداد Firebase

1. أنشئ مشروع جديد في [Firebase Console](https://console.firebase.google.com/)
2. أضف تطبيق ويب واحصل على معلومات التكوين
3. استبدل إعدادات `firebaseConfig` في ملف `firebase-config.js`
4. في قسم Authentication، فعّل خدمة البريد الإلكتروني/كلمة المرور
5. في قسم Realtime Database، اضبط قواعد الأمان:

```json
{
  "rules": {
    ".read": true,
    ".write": "auth != null"
  }
}
